# 🚀 RegistrationApp_MVC

A Java-based User Registration System developed using **JSP, Servlet, JDBC, and MySQL** following the **MVC (Model-View-Controller)** architecture.

## 📌 Project Overview

This project allows users to register through a web interface. User details are validated and securely stored in a MySQL database using JDBC. After successful registration, users are redirected to a success page; otherwise, an error page is displayed.

---

## ✨ Features

* User Registration Form
* JSP & Servlet Based MVC Architecture
* JDBC Connectivity
* MySQL Database Integration
* Session Management
* Success & Failure Pages
* PreparedStatement for Secure Database Operations

## 🚀 How to Run

1. Clone the repository.
2. Import the project into Eclipse.
3. Configure Apache Tomcat.
4. Create the MySQL database.
5. Update database credentials in `jdbcutill.java`.
6. Run the project on Tomcat.
